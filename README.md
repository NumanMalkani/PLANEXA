# Planexa - Productivity Tracking Application

This is a functional prototype for **Planexa**, a productivity tracking app built with React, Tailwind CSS, and Recharts.

## Getting Started in VS Code

To run this project locally, follow these steps:

### Prerequisites

- **Node.js**: Ensure you have Node.js installed (version 18 or higher is recommended).
- **VS Code**: The project is optimized for development in Visual Studio Code.

### Installation

1. **Extract the project**: Unzip the downloaded project files into a folder.
2. **Open in VS Code**: Open the folder in VS Code (`File > Open Folder...`).
3. **Open Terminal**: Open the integrated terminal in VS Code (`Ctrl + ` ` or `View > Terminal`).
4. **Install Dependencies**: Run the following command to install all necessary packages:
   ```bash
   npm install
   ```

### Running the App

To start the development server, run:
```bash
npm run dev
```
The app will be available at `http://localhost:3000` (or the port specified in your terminal).

## Key Features

- **Hamburger Menu**: Fully responsive navigation that works on mobile and desktop.
- **Dynamic Metrics**: Tracking for "Focus Velocity", "Flow Efficiency", and "Daily Pulse" only activates once you click a task to start a session.
- **Dark Mode**: Toggle between Light and Dark themes with smooth transitions.
- **Interactive Charts**: Visualize your productivity distribution and progression over time.
- **Session Management**: Start and stop focus sessions to log your activities.

## Technologies Used

- **React 18**: Frontend library.
- **Tailwind CSS 4**: Functional utility-first styling.
- **Motion**: Fluid animations and transitions.
- **Recharts**: Interactive data visualizations.
- **Lucide-react**: Modern icon system.
